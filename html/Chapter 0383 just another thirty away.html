Chapter 0383 "just another thirty away"

From here on out the *fam groups and formulas will be used to find prime numbers. Turn the coin over once again and use all the numbers to eliminate non-primes.

The next eight charts and XLSX files will focus on primes. Within each *fam the PS are a distance thirty (30) away: "just another thirty away". Each PS (Prime Suspect) and each possible prime is "just another thirty away" from the next or previous PS within a specific *fam. Some of this is rehash , but this is the turn point in the data.

Each chart has the same headings.

  1. "ps" the starting value , then +30 for each next value
  2. start equals the *fam
  3. Level 1 ; the PSL for the *fam
  4. The big matrix with green cells the points where the (PS / *fam) is an integer
  5. The heading boxes:
    1. The *fam
    2. The digits are the initial value that solves PS for the *fam. PS 91 = *fam07 times 13
    3. and +30 , to get to the next position: "just another thirty away". 13 goes to 43
    4. L? is the starting PSL plus some value (the *fam) to get to the next PSL

Let "s dissect the chart in smaller pieces. The same data is true for all charts.

The "ps" column:

The PS starts at one (1) and thirty is added to get to the next PS.

The "start" column:

p>This is the value equal to the *fam for the formula PSL = (PS - *fam) / 30.

The "level 1" column:

This is the PSL value. The title of this column will change with the *fam name in each file and chart.

The columns with the *fam name show the calculations done with the PSL = (PS - *fam) / 30.

The last column "Factor"

This shows TRUE for integer factors painted green. The value in the green cell and the heading are factors for the PS. FALSE if a row of non-integer values.
If it is FALSE then the PS is a prime number.

Inside the matrix are green cells with integer values.
It is the solution of the "1st factor" the top cell in the heading , and the value in the green cell.
Note the *fam in question will always have the value of one (1) under its heading.
And , the value of the *fam will reside underneath the heading *fam01.

Look at PS = 91 ; 91 = 7 x 13. This is a mapping of the PS , and factors of PS , and PSL for each *fam.

The final heading is the rule for the PS solution by this *fam.

The rule for *fam07 is as follows.

Start at 13 and add 30.

The next values will be 43 , 73 , 103.

Then , L3 (PSL = 3) and add 7.

The next PSL will be 10 , 17 , 24.
